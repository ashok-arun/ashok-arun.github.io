During the summer of 2022 I interned at Amazon Web Services as a Software development Engineer focused in machine learning. I was assigned to the client services team which focused on creating the search machine learning models for the RePost website. I was asked to improve the current machine learning models through data manipulation. I did this in a 3 step process which focused on noise reduction of the data, merging two different data sets, and developing an efficient splitting algorithm for the models.
